# ESO-Guild-Bank-Management

A React + Material UI web app for tracking Elder Scrolls Online guild bank gold flow.

## Features

- Guest mode (no account required) with local transaction tracking
- Log deposits, withdrawals, and sales tax income
- Optional notes for each log (editable and removable later)
- Edit/delete all log entries
- Daily, weekly (custom start date), monthly, and overall stats
- Grand total gold calculation for each stats period
- Optional sign up / log in in the top-right
- Logged-in guild profile sidebar on the right with create/edit/delete/select
- Guest data migration to account on login/signup, including prompt when account already has data

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```
